# JMacLabs — Slide Templates

Five sample 1920×1080 slide layouts. Each slide auto-scales to fit the viewport.

| File | Layout | Purpose |
| --- | --- | --- |
| `01-title.html` | Title | Keynote opening, podcast card |
| `02-section.html` | Section divider | Big number + chapter title |
| `03-content.html` | 3-column content | The workhorse (three ideas, one signal) |
| `04-quote.html` | Big quote | Memorable moment / pull quote |
| `05-closing.html` | Closing / contact | Thank you + links |

Palette: dark neutrals + lime signal + violet structure. One accent dominates per slide. Fraunces display, IBM Plex Sans body.
